# Tim Varley's Personal Website

A personal portfolio and blog **automatically generated by autonomous AI agents**, built with Astro and Tailwind CSS, featuring writings on AI, software engineering, and technology, along with solutions to Project Euler problems and code samples.

This website serves as a living demonstration of **100% AI-native development**. Every component, style, and architectural decision was executed by autonomous AI agents acting on high-level directives from senior engineering patterns. For more details on the AI-driven development process, see the [project overview](https://tvarley.github.io/projects/tvarley-github-io/).

## Development

**Node.js >= 22.12.0 required** (Astro 6).

```bash
npm install --legacy-peer-deps
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Deployment

This site is automatically deployed to GitHub Pages on every push to the `main` branch.

[![Deploy to GitHub Pages](https://github.com/tvarley/tvarley.github.io/actions/workflows/deploy.yml/badge.svg)](https://github.com/tvarley/tvarley.github.io/actions/workflows/deploy.yml)
