# Version 1.0.3 - Project Euler Expansion

Minor release expanding the Project Euler collection with 10 new challenging problems and improving dark theme compatibility.

## Highlights

- **Project Euler Growth**: Added problems 31-40, covering advanced topics like coin combinations, pandigital numbers, circular primes, and truncatable primes
- **Dark Theme Polish**: Enhanced select dropdown styling with proper dark theme support and browser-specific fixes
- **Code Quality**: Systematically cleaned up redundant problem description comments from code implementations

## Technical Changes

- Implemented 10 new Project Euler problems with complete multi-language solutions (C++, Java, JavaScript, Python, Go, Rust)
- Added comprehensive CSS styling for select elements and options in dark theme
- Included browser-specific fixes for Firefox and WebKit select dropdown rendering
- Updated package dependencies and build information
- Maintained consistent code formatting and external repository linking

## UI/UX Improvements

- Select dropdown elements now properly styled for dark theme with white text on dark backgrounds
- Browser-specific CSS ensures consistent appearance across Firefox, WebKit, and other engines
- Improved visual consistency of form elements throughout the application

## Content Updates

- **New Problems Added**:
  - Problem 31: Coin sums - Dynamic programming solution for £2 coin combinations
  - Problem 32: Pandigital products - Finding pandigital number products
  - Problem 33: Digit cancelling fractions - Identifying curious fraction patterns
  - Problem 34: Digit factorials - Numbers equal to sum of their digit factorials
  - Problem 35: Circular primes - Primes that remain prime in all rotations
  - Problem 36: Double-base palindromes - Palindromic in both decimal and binary
  - Problem 37: Truncatable primes - Primes that remain prime when digits are truncated
  - Problem 38: Pandigital multiples - Largest pandigital number formed by concatenation
  - Problem 39: Integer right triangles - Triangle with integral sides and maximal solutions
  - Problem 40: Champernowne's constant - Irrational decimal fraction with digit properties
- Cleaned up redundant problem description comments from existing Euler problem code blocks
- All new problems include structured solution notes with mathematical explanations
