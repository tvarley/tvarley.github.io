# Version 1.0.11 - AI Writings, Mermaid Diagrams, and Euler Expansion

Patch release expanding the AI writing series, enabling Mermaid diagram rendering, completing Euler problem descriptions through 100, and fixing navigation and analytics issues.

## Highlights

- **Content Expansion**: Four new essays covering agent-first language choice, soft skills for agents, keeping sensitive context secret, and owning a personal AI system
- **Mermaid Support**: Client-side Mermaid so flowchart fences render as interactive SVGs instead of static code blocks
- **Euler Expansion**: Problems 76–100 now include full mathematical descriptions, LaTeX, examples, and required images
- **Reliability Fixes**: Euler Solutions filters and sort work after client-side navigation; Google Analytics measurement ID is correctly interpolated
- **Licensing Clarity**: Copyright notices on articles and clearer content licensing in the README

## Features

- **Mermaid Diagram Rendering**:
  - Client-side Mermaid integration in the site layout
  - Flowchart fences render as SVGs rather than plain code blocks
  - Hero and diagram assets for the agent stack selection article

- **Licensing and Attribution**:
  - Copyright footer on blog posts
  - README updates clarifying content licensing

## Content Additions

- **"Your Preference Is Irrelevant"** (agent-first stack selection):
  - Framework for choosing languages and stacks for agentic workflows
  - Mermaid diagrams for filters, feedback loops, and fitness criteria

- **"Agent Mannerisms: Soft Skills"**:
  - Soft skills and mannerisms that improve human–agent collaboration

- **"Gandalf's Lesson: Keep It Secret, Keep It Safe"**:
  - Why and how to withhold sensitive context from agents
  - Practical patterns for scoped, secret task definitions

- **"This Is My AI..."**:
  - Building and owning a personal AI system with memory and customization

## Project Euler

- Complete mathematical descriptions for problems 76–100
- LaTeX notation, examples, and original problem imagery
- YAML formatting fixes for multiline descriptions

## Bug Fixes

- **Euler Solutions Filters**: Event delegation so filter and sort controls work after Astro client-side navigation; default sort set to descending numeric order
- **Google Analytics**: Measurement ID passed via `define:vars` so gtag receives the real ID instead of a literal string
- **Build and Deploy**: Tailwind and workflow adjustments for stable builds under Node 22 / Astro 6

## Technical Updates

- Dependency and lockfile updates aligned with Astro 6 stack
- README refresh for Node engine requirements and install guidance
- Build info generation retained for deployment visibility

## Quality Assurance

- Verified Mermaid diagrams render from flowchart fences
- Confirmed Euler filter/sort behavior after client navigation
- Validated new writings and Euler content integration
- Ensured build stability with all changes
