      uint64_t largest_product_brute(int len)
      {
        int idx = 0;
        uint64_t answer = 0;

        while( -1 != digits[idx+len] )
        {
          uint64_t tmp = 1;
          for( int i = 0 ; i < len ; i++ ){
            tmp *= digits[idx+i];
          }

          answer = std::max(tmp,answer);
          idx++;
        }
        return answer;
      }

## Additional Notes

This problem involves finding the maximum product of 13 consecutive digits in a large 1000-digit number. The solution uses a sliding window approach to efficiently compute products of adjacent digit groups.

Both implementations include the complete 1000-digit sequence as an array of integers for easy manipulation. The C++ version uses templates for compile-time optimization of the window size, while the Ruby version uses a simple iterative approach.
